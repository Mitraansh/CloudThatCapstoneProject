# Self-Evaluation

This project is a Flower Shop capstone application with:

- FastAPI backend and SQLite database.
- Protected signup/login, chat, and image upload.
- Simple frontend pages for chat and status.
- Basic RAG and Text2SQL-like query handling.
- Tests for auth, status, chat authorization, and upload validation.

## Running the app
1. Install requirements.
2. Set `APP_SECRET_KEY` and `OPENROUTER_API_KEY` in `.env`.
3. Start the app: `python Backend/app.py`.
